# Domain

Question: What does the product language mean?

Last reviewed: 2026-06-19

## Core Concepts

| Concept    | Meaning                                                                                                                                 | Canonical ownership |
| ---------- | --------------------------------------------------------------------------------------------------------------------------------------- | ------------------- |
| Commit     | The atomic record of disciplined action. Source material for progress, journey, and presence.                                           | This file           |
| Circle     | A small private set of people who witness and support the user's process. Not a feed, follower graph, or public social network.         | This file           |
| Presence   | The feeling that Circle members are still in the process. Derived from recent activity, not chat volume.                                | This file           |
| Identity   | The user's internal shift from aspiration to self-description. It is reinforced across the product, not stored as a standalone feature. | This file           |
| Progress   | Visible evidence accumulated from repeated action. Calculated mainly from Commits.                                                      | This file           |
| Journey    | The narrative timeline of continuity, pauses, returns, and growth.                                                                      | This file           |
| Reflection | A short insight that connects action to emotion, learning, or identity.                                                                 | This file           |
| Knowledge  | Practical learning that may be saved or shared later. Supporting/future, not core MVP.                                                  | This file           |
| Support    | A lightweight Circle interaction that acknowledges another person's effort.                                                             | This file           |

## Relationships

- Commit is the root event.
- Progress is derived from Commits.
- Journey is the narrative form of Progress over time.
- Presence is derived from Circle relationships and recent Commits.
- Circle gives Progress witnesses without turning it into public performance.
- Reflection deepens the meaning of Commit and Progress.
- Identity is an emergent product outcome reinforced by language, visibility,
  and continuity.
- Knowledge is useful only when it serves action and shared practice.

## Boundaries

Gym Circle is not:

- a public social network
- a fitness content feed
- a ranking system
- a body-comparison app
- a generic habit tracker
- a chat product

## MVP Domain Priority

Core:

- Commit
- Circle
- Presence
- Progress
- Journey
- Identity reinforcement

Supporting:

- Reflection
- Support
- Profile

Future:

- Knowledge
- Notifications
- Personal records
